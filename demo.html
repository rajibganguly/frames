Form Elements

All elements used as in as html5 Text, Password, Email, URL, Number and Range. 
<input type="text" class="f-txt f-fontSize13" placeholder="Enter Text" required />
<input type="password" class="f-password f-fontSize13" placeholder="Enter Password" required />
<input type="email" class="f-email f-fontSize13" placeholder="Enter Email" required />
<input type="url" class="f-url f-fontSize13" placeholder="Enter website" required />
<input type="number" class="f-number f-fontSize13" placeholder="Enter Number" required />
<input type="color" class="f-color f-fontSize13" placeholder="Enter Color" required />
<input type="range" id="a" value="50" class="f-range f-fontSize13" />
NB: Pattern used in Text input not for Safari browser.
